# SCT_WD_2
A responsive Stopwatch Web Application built using HTML, CSS, and JavaScript. The application includes start, pause, reset, and lap time tracking functionalities with an interactive and user-friendly interface. This project demonstrates real-time timer functionality, DOM manipulation, and responsive frontend design.
